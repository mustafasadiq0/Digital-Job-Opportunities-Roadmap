# موقع "وظيفتك الرقمية 2035" - ملخص المشروع

## 🌟 نظرة عامة
تم إنشاء موقع إلكتروني تفاعلي شامل بعنوان "خارطة الطريق لأهم 20 وظيفة مستقبلية في تقنيات المعلومات والاتصالات للـ10 سنوات القادمة" باللغة العربية مع إمكانية التوسع للإنجليزية لاحقاً.

## 🔗 رابط الموقع المنشور
**الرابط المباشر:** https://ogtzgpdd.manus.space

## 📋 المحتوى والميزات

### الصفحة الرئيسية
- **العنوان الرئيسي:** وظيفتك الرقمية 2035
- **الشعار:** تصميم احترافي مع هوية بصرية متميزة
- **إحصائيات مرئية:** 20 وظيفة مستقبلية، 5 تصنيفات رئيسية، 100+ مصدر تعليمي مجاني

### التصنيفات الرئيسية (5 مجالات)
1. **الذكاء الاصطناعي** - وظائف مستقبلية في مجال الذكاء الاصطناعي والتعلم الآلي
2. **الأمن السيبراني** - وظائف حماية البيانات والأنظمة الرقمية
3. **الحوسبة السحابية** - وظائف تطوير وإدارة الحلول السحابية
4. **تطوير البرمجيات** - وظائف تطوير التطبيقات والأنظمة البرمجية
5. **علم البيانات** - وظائف تحليل البيانات واستخراج الرؤى

### الوظائف المميزة (20 وظيفة)
كل وظيفة تتضمن:
- **نظرة عامة مختصرة** عن المسمى الوظيفي
- **الشهادات المعتمدة** (مجانية ومنخفضة التكلفة)
- **المهارات التقنية والناعمة** المطلوبة
- **مقترحات لمشاريع عملية** لتقوية المهارات
- **روابط موثوقة لمصادر مجانية** (دورات، مقالات، فيديوهات)
- **تصنيف حسب المستوى** (مبتدئ، متوسط، متقدم)

### أمثلة على الوظائف المتضمنة:
- مدرب نماذج ذكاء اصطناعي
- مهندس متخصص في الذكاء الاصطناعي (Prompt Engineer)
- محلل أمن معلومات
- مطور تطبيقات سحابية
- مطور ويب
- عالم بيانات
- مهندس تعلم الآلة
- وغيرها...

### نصيحة الأسبوع
قسم مخصص لعرض مصدر تعليمي مجاني جديد أسبوعياً، يتضمن:
- **العنوان والوصف**
- **مدة الدورة**
- **المستوى المطلوب**
- **مقدم الخدمة**
- **رابط مباشر للمصدر**

## 🎨 التصميم والتقنيات

### التصميم
- **نمط التصميم:** بسيط، أنيق، وسهل التصفح
- **الألوان:** لوحة ألوان هادئة بدرجات الأزرق والأبيض
- **الخطوط:** خطوط عربية حديثة (Cairo و Noto Kufi Arabic)
- **الأيقونات:** أيقونات مخصصة لكل تصنيف ومهارة
- **الاستجابة:** متوافق مع جميع الأجهزة (موبايل، تابلت، ديسكتوب)

### التقنيات المستخدمة
- **Frontend:** React.js مع Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **Deployment:** Manus Cloud Platform
